## Project Topic
**Demon Slayer Merchandise Shop** is an e-commerce website dedicated to *Kimetsu no Yaiba (Demon Slayer)* fans.  
It features a range of anime-inspired clothing, accessories, and collectibles for men, women, and children.  
The website delivers a modern, responsive, and interactive shopping experience.

---

## Key Features

### **Design and User Experience**
- Fully responsive layout for all devices (desktop, tablet, mobile)  
- Light/Dark mode toggle with Local Storage persistence  
- Clean anime-themed interface and modern typography  
- Interactive product galleries and category-based browsing  

### **Enhanced JavaScript Functionality**
- Shopping cart system available across all pages  
- Product search and category filtering  
- Login and registration form validation  
- FAQ accordion system  
- Image hover effects and subtle animations  
- Animated statistics counters  

### **E-commerce Features**
- Multi-category product catalog (Men, Women, Kids, New Arrivals)  
- Add-to-cart and remove-from-cart item management  
- User authentication simulation  
- Product search and category filters  

### **Technical Features**
- **jQuery** used for DOM manipulation and event handling  
- **Local Storage** used for saving user preferences (themes, cart items, etc.)  
- Responsive **navigation bar** with hamburger menu  
- Consistent design and functionality across all pages
